# 🚀 Fake Max
> Fake application imitating the national messenger Max.

![Демонстрация](ссылка_на_gif.gif)



## 📌 Описание
The application creates the appearance of the presence of this application on your Android device, which repels suspicions about you.

## 🛠️ Технологии
- [Python](https://python.org)
- [kivy](https://kivy.org)
- [buildozer](https://buildozer.readthedocs.io)
